Instalar en celular o computadora

Tu navegador permite instalar la app directamente:

iPhone / iPad (Safari)

Tocá Compartir ↗ abajo → Agregar a pantalla de inicio

Android (Chrome)

Menú ⋮ → Instalar aplicación o Agregar a pantalla principal

Computadora (Chrome / Edge)

Ícono de instalar en la barra de direcciones, o menú → Instalar Nexa Soluciones

Para que aparezca "Instalar", la app tiene que abrirse desde un link web (https), no como archivo local.
